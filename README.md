## Polindrom Kontrol Uygulaması
Bu proje, kullanıcının girdiği kelime veya cümlelerin palindrom olup olmadığını kontrol eden, tamamen HTML, CSS ve JavaScript ile geliştirilmiş basit ama etkili bir web uygulamasıdır.

## Palindrom Nedir?
Palindrom; tersten okunduğunda da anlamını veya yazımını kaybetmeyen kelimelere ve cümlelere verilen addır.

## Projenin Amacı

Bu uygulama, JavaScript string işlemleri, DOM manipülasyonu ve regex kullanımını uygulamalı olarak öğrenmek amacıyla geliştirilmiştir.  
Aynı zamanda modern CSS ile basit bir UI tasarımı uygulama pratiği yapılmıştır.

##  Özellikler

- Girilen metnin palindrom olup olmadığını gerçek zamanlı kontrol eder  
- Büyük/küçük harf ayrımını tamamen yok sayar  
- Boşlukları, noktalama işaretlerini ve özel karakterleri filtreler  
- Sayılar ile yapılan kontrolleri destekler  
- Enter tuşu ile kontrol edebilme özelliği  
- Mobil ve masaüstü cihazlarla tamamen uyumlu responsive tasarım  
- Minimal, şık ve kullanıcı dostu arayüz  
- Anında geri bildirim sağlayan mesaj sistemi

## Kullanılan Teknolojiler

- HTML5 – Sayfa yapısı
- CSS3 – Modern tasarım ve mobil uyum
- JavaScript (Vanilla) – Palindrom algoritması ve DOM yönetimi
